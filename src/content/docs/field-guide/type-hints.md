## Что это

Аннотации типов описывают, какие значения ожидает код и что он возвращает. Python обычно не проверяет их при запуске, но редактор, анализатор типов и читатель получают явный контракт.

```python
def risk_label(score: int) -> str:
    return "высокий риск" if score >= 7 else "низкий риск"
```

Здесь `score: int` описывает параметр, а `-> str` — результат. `-> None` означает, что функция выполняет действие и не возвращает полезного значения.

## Типы коллекций

Тип элемента записывается в квадратных скобках:

```python
names: list[str] = ["Алина", "Никита"]
scores: dict[str, int] = {"Алина": 7}
pair: tuple[str, str] = ("report-a", "report-b")
events: tuple[Evidence, ...]
```

Многоточие в `tuple[Evidence, ...]` означает кортеж любой длины, в котором каждый элемент — `Evidence`.

## `Any`, `None` и объединения

Функция загрузки временно работает с `dict[str, Any]`, потому что JSON-словарь может содержать значения разных типов. `Any` отключает сведения о конкретном типе; после чтения лучше как можно быстрее преобразовать данные в понятную модель.

```python
from typing import Any

def load_record(data: dict[str, Any]) -> Evidence:
    ...
```

Запись `str | None` означает «строка или отсутствие значения». Перед использованием как строки нужно проверить `None`.

## `Self` в фабричном методе

`Self` означает объект текущего класса. Это подходит методу, который создаёт экземпляр через `cls`:

```python
from typing import Any, Self

class Evidence:
    @classmethod
    def from_dict(cls, data: dict[str, Any]) -> Self:
        return cls(title=str(data["title"]))
```

Если от `Evidence` появится подкласс, `Self` сохранит более точный тип результата.

## Типичные ловушки

- Аннотация не превращает значение автоматически: `score: int = "7"` по-прежнему сохраняет строку.
- Не используйте `Any` внутри всей программы только потому, что входной JSON неоднороден.
- Тип `list[str]` не принимает `None`, если это не указано явно.
- Аннотации должны помогать читать границы функций и моделей, а не дублировать очевидную локальную переменную на каждой строке.
