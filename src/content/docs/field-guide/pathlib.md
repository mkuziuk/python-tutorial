## Что это

`pathlib.Path` представляет путь к файлу или папке как объект с понятными методами.

## Когда использовать

Используйте `Path`, когда программа читает файлы, создаёт папки, ищет данные или должна работать на Windows, macOS и Linux.

```python
from pathlib import Path

data_dir = Path("data")
for path in data_dir.rglob("*"):
    print(path)
```

Пример предполагает существующую папку `data`. `rglob()` возвращает поток
объектов `Path`; конкретный порядок зависит от файловой системы, поэтому такой
вывод годится для обзора, но не для воспроизводимого отчёта.

## Мини-пример

```python
from pathlib import Path

report = Path("out") / "manifest.json"
report.parent.mkdir(exist_ok=True)
report.write_text("{}", encoding="utf-8")
print(report.exists(), report.read_text(encoding="utf-8"))
```

```text
True {}
```

Побочный эффект — файл `out/manifest.json` относительно текущей рабочей папки.
Вывод проверяет и наличие файла, и точное текстовое содержимое. Для обхода уже
созданного дерева затем можно отфильтровать нужные расширения.

## Ещё примеры

Ищите только файлы нужного типа и отделяйте имя, расширение и размер:

```python
from pathlib import Path

for path in Path("data").rglob("*.txt"):
    if path.is_file():
        print(path.name, path.suffix, path.stat().st_size)
```

Каждая строка имеет форму `имя, расширение, размер в байтах`; значения зависят
от содержимого `data`. Проверка `is_file()` не пропускает каталоги с суффиксом
`.txt`, а размер относится к байтам на диске, не к числу символов.

Путь можно сделать относительным к корню расследования, чтобы отчёт не зависел от имени домашней папки:

```python
root = Path("data")
path = root / "archive" / "timeline.txt"

print(path.relative_to(root))  # archive/timeline.txt
```

```text
archive/timeline.txt
```

Результат не содержит абсолютную домашнюю папку, поэтому его можно безопасно
сохранить в переносимом отчёте.

## Типичные ловушки

- `Path("data")` зависит от текущей рабочей папки. Для файлов рядом со скриптом используйте `Path(__file__).parent`.
- `relative_to()` работает только когда путь действительно лежит внутри указанного корня.
- `rglob("*")` находит и файлы, и папки. Проверяйте `path.is_file()`.
- Не склеивайте пути вручную через строки, используйте оператор `/`.
