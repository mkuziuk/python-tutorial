## Что это

`str` хранит текст. В наших делах строка часто является сырой уликой: письмо, лог, имя файла, строка отчета.

## Когда использовать

Используйте `str`, когда нужно хранить или подготовить текст: привести к нижнему регистру, найти слова, проверить окончание файла, собрать сообщение для отчета.

```python
text = "След оставлен ночью."
clean = text.lower()
```

## Мини-пример

```python
message = "Пауза была слишком длинной."
words = message.lower().replace(".", "").split()
print(words)
```

## Типичные ловушки

- `split()` режет только по пробелам и не убирает пунктуацию.
- `lower()` не меняет исходную строку, а возвращает новую.
- Индексы начинаются с нуля.
