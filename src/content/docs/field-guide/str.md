## Что это

`str` хранит текст. В наших делах строка часто является сырой уликой: письмо, лог, имя файла, строка отчёта.

## Когда использовать

Используйте `str`, когда нужно хранить или подготовить текст: привести к нижнему регистру, найти слова, проверить окончание файла, собрать сообщение для отчёта.

```python
text = "След оставлен ночью."
clean = text.lower()
```

## Мини-пример

```python
message = "Пауза была слишком длинной."
words = message.lower().replace(".", "").split()
print(words)
```

## Ещё примеры

Проверить формат строки можно без регулярных выражений:

```python
filename = "report_2026.txt"

if filename.startswith("report_") and filename.endswith(".txt"):
    print("похож на текстовый отчёт")
```

Для читаемого сообщения подставляйте значения через f-строку. Выражение внутри фигурных скобок вычисляется в момент создания новой строки:

```python
author = "Морозова"
score = 0.77
summary = f"Кандидат: {author}; сходство: {score:.0%}"
print(summary)  # Кандидат: Морозова; сходство: 77%
```

## Типичные ловушки

- `split()` без аргумента разделяет по пробельным символам — пробелам, табуляции и переносам строки, — но не убирает пунктуацию.
- `lower()` не меняет исходную строку, а возвращает новую.
- `replace()` заменяет точное сочетание символов и тоже возвращает новую строку.
- Индексы начинаются с нуля.
