## Что это

Функция - именованный кусок программы. Она получает данные, делает один понятный шаг и может вернуть результат.

## Когда использовать

Выносите код в функцию, когда шаг можно назвать глаголом: прочитать файл, очистить слова, сравнить профили, напечатать отчет.

```python
def normalize_words(text):
    return text.lower().split()


words = normalize_words("След рядом")
print(words)
```

## Мини-пример

```python
def risk_label(score):
    if score >= 7:
        return "высокий риск"
    if score >= 3:
        return "проверить вручную"
    return "низкий риск"


print(risk_label(5))
```

## Типичные ловушки

- Функция должна делать один ясный шаг, а не весь проект сразу.
- `return` отдает результат наружу; `print()` только показывает текст в терминале.
- Имя функции должно объяснять действие: `build_profile()` лучше, чем `process()`.
- Если функция меняет файл или печатает таблицу, не смешивайте это с подсчетами без нужды.
