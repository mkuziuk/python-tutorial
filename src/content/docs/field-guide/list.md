## Что это

`list` хранит несколько значений в заданном порядке. В расследованиях это удобно для слов текста, строк файла, найденных событий или кандидатов в отчете.

## Когда использовать

Берите список, если порядок важен или элементы нужно проходить один за другим.

```python
words = ["след", "пауза", "запись"]
for word in words:
    print(word)
```

## Мини-пример

```python
scores = [0.77, 0.52, 0.45]
scores.sort(reverse=True)
print(scores[0])
```

## Типичные ловушки

- Методы вроде `.sort()` меняют список на месте.
- Копия через `other = words` не создает новый список.
- Пустой список нельзя читать по индексу `0`.
