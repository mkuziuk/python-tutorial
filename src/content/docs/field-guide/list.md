## Что это

`list` хранит несколько значений в заданном порядке. В расследованиях это удобно для слов текста, строк файла, найденных событий или кандидатов в отчёте.

## Когда использовать

Берите список, если порядок важен или элементы нужно проходить один за другим.

```python
words = ["след", "пауза", "запись"]
for word in words:
    print(word)
```

## Мини-пример

```python
scores = [0.77, 0.52, 0.45]
scores.sort(reverse=True)
print(scores[0])
```

## Ещё примеры

Список можно постепенно собрать, а затем оставить только нужные элементы:

```python
events = []
events.append("вход 22:14")
events.append("ошибка чтения")
events.append("выход 22:19")

night_events = [event for event in events if "22:" in event]
print(night_events)
```

Если исходный порядок нужно сохранить, используйте `sorted()`: она возвращает новый список, в отличие от `.sort()`.

```python
scores = [0.52, 0.77, 0.45]
ranked = sorted(scores, reverse=True)

print(scores)  # исходный список не изменился
print(ranked)
```

## Типичные ловушки

- Методы вроде `.sort()` меняют список на месте.
- Копия через `other = words` не создаёт новый список.
- Срез `words[:]` создаёт новый внешний список, но вложенные изменяемые объекты останутся общими.
- Пустой список нельзя читать по индексу `0`.
