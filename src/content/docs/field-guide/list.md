## Что это

`list` хранит несколько значений в заданном порядке. В расследованиях это удобно для слов текста, строк файла, найденных событий или кандидатов в отчёте.

## Когда использовать

Берите список, если порядок важен или элементы нужно проходить один за другим.

```python
words = ["след", "пауза", "запись"]
for word in words:
    print(word)
```

```text
след
пауза
запись
```

Вывод сохраняет порядок элементов: цикл читает список слева направо. Когда
важен не весь порядок, а лучший числовой результат, список можно отсортировать.

## Мини-пример

```python
scores = [0.77, 0.52, 0.45]
scores.sort(reverse=True)
print(scores[0])
```

```text
0.77
```

`.sort(reverse=True)` изменяет сам `scores`, после чего индекс `0` содержит
максимальную оценку. Если исходная последовательность понадобится дальше,
нужно собрать отдельный отфильтрованный или отсортированный список.

## Ещё примеры

Список можно постепенно собрать, а затем оставить только нужные элементы:

```python
events = []
events.append("вход 22:14")
events.append("ошибка чтения")
events.append("выход 22:19")

night_events = [event for event in events if "22:" in event]
print(night_events)
```

```text
['вход 22:14', 'выход 22:19']
```

`events` содержит три строки, а `night_events` — новый список из двух строк с
маркером `22:`. Исходный список сохранился, поэтому тот же принцип подходит,
когда порядок исходных оценок нельзя менять.

Если исходный порядок нужно сохранить, используйте `sorted()`: она возвращает новый список, в отличие от `.sort()`.

```python
scores = [0.52, 0.77, 0.45]
ranked = sorted(scores, reverse=True)

print(scores)
print(ranked)
```

```text
[0.52, 0.77, 0.45]
[0.77, 0.52, 0.45]
```

Две строки подтверждают границу изменения: `scores` остался в исходном
порядке, а `ranked` получил новый порядок.

## Типичные ловушки

- Методы вроде `.sort()` меняют список на месте.
- Копия через `other = words` не создаёт новый список.
- Срез `words[:]` создаёт новый внешний список, но вложенные изменяемые объекты останутся общими.
- Пустой список нельзя читать по индексу `0`.
