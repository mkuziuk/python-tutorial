## Что это

Регулярное выражение - это шаблон для поиска текста. В первом деле оно помогает достать слова без приклеенной пунктуации.

## Когда использовать

Используйте `re`, если простого `.split()` мало: нужно найти слова, даты, номера, теги или повторяющиеся текстовые формы.

```python
import re

word_re = re.compile(r"[а-яё]+", re.IGNORECASE)
words = word_re.findall("След есть. След рядом!")
print(words)
```

## Мини-пример

```python
import re

pattern = re.compile(r"[а-яё]+", re.IGNORECASE)
print(pattern.findall("Архив-17: новая Запись."))
```

Результат:

```text
['Архив', 'новая', 'Запись']
```

## Типичные ловушки

- `.` в regex означает почти любой символ, а не точку. Для настоящей точки пишут `\.`
- `+` означает "один или больше", поэтому `[а-яё]+` склеивает соседние буквы в слово.
- `r"..."` не обязательно всегда, но помогает не запутаться с обратными слешами.
- Regex полезен, но не должен заменять нормальный парсер, если формат данных сложный.
