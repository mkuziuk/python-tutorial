## Что это

`Counter` из модуля `collections` — словарь, который хранит пары `элемент → количество появлений`.

## Когда использовать

Используйте `Counter`, когда нужно быстро понять, что повторяется чаще: слова, символы, расширения файлов или n-граммы.

Допустим, после токенизации короткой записки получен список строк. Следующий этап должен превратить его в счётчик формы `слово → количество`, чтобы частоты можно было сравнивать.

```python
from collections import Counter

words = ["след", "архив", "след"]
word_counts = Counter(words)
print(word_counts)
```

Результат:

```text
Counter({'след': 2, 'архив': 1})
```

Ключи счётчика — исходные строки, значения — целые количества. Вывод подтверждает, что слово `след` встретилось два раза, а `архив` — один. На реальном тексте список `words` обычно получают после приведения регистра и токенизации; без нормализации варианты `След` и `след` останутся разными ключами.

## Мини-пример

`most_common(n)` выбирает `n` наиболее частых элементов и возвращает список кортежей формы `(элемент, количество)`. Проверим первые три буквы по частоте:

```python
from collections import Counter

letters = Counter("расследование")
print(letters.most_common(3))
```

Результат:

```text
[('а', 2), ('с', 2), ('е', 2)]
```

Все три буквы встречаются по два раза. При равной частоте `Counter` сохраняет порядок первого появления, поэтому `а`, `с` и `е` расположены именно так. Мы получили компактный список лидеров; если нужны все частоты, можно вызвать `most_common()` без аргумента.

## Ещё примеры

Счётчик можно пополнять частями — это удобно, когда слова поступают из нескольких файлов. Каждый вызов `update()` добавляет новые наблюдения к уже накопленным значениям:

```python
from collections import Counter

word_counts = Counter()
word_counts.update(["архив", "ночь", "архив"])
word_counts.update(["след", "ночь"])

for word, count in word_counts.most_common():
    print(word, count)
```

Результат:

```text
архив 2
ночь 2
след 1
```

После двух обновлений `word_counts` имеет форму `слово → суммарное количество во всех обработанных частях`. Сумма равна пяти наблюдениям. Равные количества `архив` и `ночь` снова упорядочены по первому появлению.

Мы проверили абсолютные количества. Чтобы сравнивать тексты разной длины, нужно перейти от количества к доле — разделить частоту выбранного слова на общее число наблюдений:

```python
total = word_counts.total()
night_share = word_counts["ночь"] / total if total else 0
print(f"доля слова 'ночь': {night_share:.1%}")
```

Результат:

```text
доля слова 'ночь': 40.0%
```

`total` равен `5`, а счётчик для слова `ночь` равен `2`, поэтому доля составляет `2 / 5 = 40%`. Условие `if total else 0` определяет границу для пустого счётчика и не допускает деления на ноль. Такая доля сравнима между текстами разного размера, но на коротком учебном списке одно слово заметно меняет результат.

## Типичные ловушки

- `Counter` считает точные значения: `"След"` и `"след"` разные строки.
- Обращение к отсутствующему элементу возвращает `0`, а не вызывает `KeyError`.
- `most_common()` сортирует по частоте, но равные частоты могут идти в порядке первого появления.
- Для долей и процентов все равно нужно делить на общее количество элементов.
