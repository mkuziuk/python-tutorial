## Что это

`tuple` - неизменяемая последовательность. В расследованиях кортеж удобен, когда несколько значений вместе образуют один признак.

## Когда использовать

Используйте кортеж, если порядок важен, а менять элементы не нужно. Например, n-грамма из трёх слов может быть кортежем.

```python
trigram = ("ночной", "архив", "молчит")
print(trigram)
```

```text
('ночной', 'архив', 'молчит')
```

Кортеж хранит ровно три позиции в заданном порядке. Чтобы построить такие
признаки для всего текста, нужно превратить каждое окно из трёх слов в кортеж.

## Мини-пример

```python
words = ["ночной", "архив", "молчит", "снова"]
ngrams = [tuple(words[index:index + 3]) for index in range(len(words) - 2)]
print(ngrams)
```

```text
[('ночной', 'архив', 'молчит'), ('архив', 'молчит', 'снова')]
```

Четыре слова дают два перекрывающихся окна длины три; `ngrams` имеет форму
«список кортежей слов». Отдельный кортеж можно распаковать, если у каждой
позиции свой смысл.

## Ещё примеры

Кортеж удобно распаковать в несколько осмысленных имён:

```python
event = ("2026-03-14T22:14:00Z", "login", "archive-room")
timestamp, action, place = event

print(action, place)
```

```text
login archive-room
```

Распаковка связала вторую и третью позиции с осмысленными именами; временная
метка осталась в `timestamp`. Когда значение нужно находить не по позиции, а
по составному идентификатору, кортеж подходит как ключ словаря.

Кортеж также может быть составным ключом словаря — например, когда важна именно пара файлов:

```python
similarity = {
    ("draft.txt", "report.txt"): 0.82,
    ("notes.txt", "report.txt"): 0.31,
}

print(similarity[("draft.txt", "report.txt")])
```

```text
0.82
```

Вывод подтверждает, что пара имён образует один ключ и ведёт к одной оценке.
Порядок внутри пары значим: обратный кортеж был бы другим ключом.

## Типичные ловушки

- Кортеж из одного элемента пишется с запятой: `("след",)`.
- При распаковке число имён должно совпадать с числом элементов.
- Кортеж нельзя изменить на месте. Если нужен новый вариант, создайте новый кортеж.
- Список нельзя положить в `set`, а кортеж из строк можно.
