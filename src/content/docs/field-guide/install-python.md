## Что нужно

Для расследований Части I нужен Python 3.13 или новее. Сначала проверьте версию:

```powershell
py -3 --version
```

```bash
python3 --version
```

Обе команды должны показать версию не ниже 3.13. На Windows `py -3` выбирает установленный Python 3, не привязывая окружение к конкретному минорному выпуску.

## Где запускать команды

Сначала распакуйте архив расследования. Затем откройте получившуюся папку `case-0N` в обычном файловом менеджере:

- **Windows:** откройте папку в Проводнике, щёлкните правой кнопкой по свободному месту внутри неё и выберите **Открыть в терминале**;
- **macOS:** откройте папку в Finder, щёлкните по ней правой кнопкой и выберите **Службы → Новый терминал по адресу папки**;
- **Linux:** откройте папку в файловом менеджере, щёлкните правой кнопкой по свободному месту и выберите **Открыть в терминале**.

Название пункта зависит от версии системы и файлового менеджера: например,
вместо «Открыть в терминале» может быть «Открыть PowerShell здесь» или
«Open Terminal Here». Важно, чтобы терминал открылся именно в папке
расследования Части I, где лежат `requirements.txt`, стартовый `.py` и папка
`data/`.

## Виртуальная среда

Виртуальная среда — отдельная папка с Python только для одного расследования. Так разные проекты не мешают друг другу.

Windows PowerShell:

```powershell
py -3 -m venv .venv
.\.venv\Scripts\Activate.ps1
```

macOS или Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

После активации в начале строки терминала обычно появляется `(.venv)`, но вид
приглашения можно настроить. Надёжнее проверить путь запущенного интерпретатора:

```bash
python -c "import sys; print(sys.executable)"
```

На Windows путь должен оканчиваться примерно на
`case-0N\.venv\Scripts\python.exe`, а на macOS и Linux — на
`case-0N/.venv/bin/python`. Это подтверждает, что команды используют Python
именно этого расследования. Все четыре проекта Части I работают только со
стандартной библиотекой, поэтому устанавливать дополнительные пакеты не нужно.
Файл `requirements.txt` оставлен как явная отметка об отсутствии внешних
зависимостей.
