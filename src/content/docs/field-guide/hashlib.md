## Что это

`hashlib` считает короткий цифровой отпечаток данных. Если два файла имеют одинаковый SHA-256, их содержимое практически наверняка совпадает.

## Когда использовать

Используйте хеш, когда имя файла ненадежно: для поиска дублей, проверки изменений и сравнения версий архива.

```python
import hashlib
from pathlib import Path

digest = hashlib.sha256(Path("note.txt").read_bytes()).hexdigest()
```

## Мини-пример

```python
import hashlib

data = b"case note"
print(hashlib.sha256(data).hexdigest())
```

## Типичные ловушки

- Хеш не говорит, что внутри файла, он говорит только про байты.
- Для больших файлов лучше читать блоками, а не одним `read_bytes()`.
- Хеш меняется даже от одного лишнего пробела.
