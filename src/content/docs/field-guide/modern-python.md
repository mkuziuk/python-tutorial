## Версия

Учебник ориентируется на Python 3.14 и новее. На момент первой версии сайта актуальная bugfix-ветка CPython - 3.14.x.

В проектах есть проверка:

```bash
pnpm check:python
```

Она падает, если локальный `python3` старее 3.14.

## Стиль кода

В примерах используется современный, но не декоративный Python:

- встроенные обобщенные типы: `list[str]`, `dict[str, object]`, `set[str]`;
- `pathlib.Path` для путей;
- f-строки для форматирования;
- `Counter` из стандартной библиотеки для частот;
- маленькие функции с понятными именами.

Мы не используем устаревшие API ради совместимости со старыми версиями Python. Если старый прием встречается в интернете, он должен быть переписан перед попаданием в учебник.

## Библиотеки

Первая арка использует одну стороннюю библиотеку: `rich==15.0.0`. Она отвечает только за красивую таблицу в терминале. Анализ данных остается на стандартной библиотеке Python.

Когда проекту действительно понадобится библиотека, правило такое:

- брать актуальную стабильную версию;
- проверять документацию перед добавлением;
- фиксировать, в каком деле она используется;
- не показывать функции и параметры, которые уже помечены как deprecated.
