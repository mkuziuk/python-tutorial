## Канонический вывод

**Установленный факт.** Более сильные модели превосходят `DummyRegressor` при случайном разбиении, однако ошибки неоднородны по цене и географии. На географической тестовой выборке качество падает, а верхнее ограничение целевой переменной искажает поведение у границы.

**Поддержанная интерпретация.** Случайное разбиение поддерживает более узкое заявление об интерполяции среди похожих областей. Оно не проверяет обещание переноса в действительно новый регион.

**Что не доказано.** Географический разрыв не означает, что модель бесполезна в каждом месте, и не выявляет одну универсальную причину ошибок.

**Ограничения.** Набор агрегирован по кварталам и относится к определённому историческому периоду; цена выражена в сотнях тысяч долларов и обрезана сверху. Учебные регионы — приближение к сценарию внедрения.

**Рекомендованное действие.** Определить реальные регионы применения, сформировать независимый региональный тест и публиковать MAE/RMSE вместе с картой и сегментными ошибками.

## Что важно унести

MAE отвечает на вопрос о типичной абсолютной ошибке, RMSE сильнее штрафует крупные промахи, а R² сравнивает модель с вариативностью цели. Ни одна из этих метрик не показывает, **где** возникают ошибки; для этого нужны остатки и срезы.
