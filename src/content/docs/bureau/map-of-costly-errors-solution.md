## Канонический вывод

**Установленный факт.** Случайный лес превосходит `DummyRegressor` при случайном
разбиении. Его RMSE составляет примерно 0,50, то есть около 50 000 долларов.
На holdout из невиданных географических ячеек RMSE возрастает примерно до
0,66, или 66 000 долларов. Остатки, карта и срезы показывают, что ошибки
неоднородны по цене и географии.

**Поддержанная интерпретация.** Случайное разбиение поддерживает более узкое
заявление об интерполяции среди похожих областей. Нулевое пересечение
`region_id` в групповом split создаёт более строгую проверку переноса, и
ухудшение RMSE показывает, что случайная оценка переоценивает поддержку
обещания о новых регионах.

**Что не доказано.** Один географический split не означает, что модель
бесполезна в каждом месте, не даёт доверительного интервала и не выявляет одну
универсальную причину ошибок. Пространственное скопление промахов не является
причинным объяснением.

**Ограничения.** Набор агрегирован по кварталам переписи 1990 года; цена
выражена в сотнях тысяч долларов и более чем для 900 строк обрезана сверху на
`5.00001`. Ячейки 0,5° × 0,5° и граница north/south по 37° — заранее заданные
учебные приближения, а не реальные области внедрения.

**Рекомендованное действие.** Определить реальные регионы применения,
сформировать независимый региональный тест и публиковать MAE/RMSE вместе с
картой и сегментными ошибками, всегда показывая `n` для каждого среза.

Эти числа — наблюдаемый результат фиксированного снимка и фиксированных
разбиений. Они поддерживают более осторожное решение о дополнительной
региональной проверке, но не заменяют данные текущего рынка. Полный ход
вычислений и непосредственные интерпретации после каждой таблицы и визуализации
сохранены в тетради решения.

## Что важно унести

MAE отвечает на вопрос о типичной абсолютной ошибке, RMSE сильнее штрафует крупные промахи, а R² сравнивает модель с вариативностью цели. Ни одна из этих метрик не показывает, **где** возникают ошибки; для этого нужны остатки и срезы.
