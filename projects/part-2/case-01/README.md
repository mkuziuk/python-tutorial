# Дело II-01. Ирисовый экзамен

Первое задание в Бюро аналитических расследований: проверить демонстрацию «Компаса», в которой классификатор безошибочно определил вид ириса. Вы воспроизведёте честную проверку модели и отделите качество на обучающих данных от способности работать с новыми наблюдениями.

## Результат дела

За 3–4 часа вы:

- превратите встроенный набор Iris в `DataFrame` и исследуете признаки;
- сформулируете задачу через `X` и `y`;
- сделаете стратифицированное разделение данных;
- сравните `DummyClassifier` и масштабируемый k-NN;
- прочитаете accuracy и матрицу ошибок;
- оформите пятичастную аудиторскую записку без необоснованного обвинения.

## Подготовка

Нужен Python 3.12 или 3.13. В каталоге дела выполните:

```bash
python3 -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
python -m pip install -r requirements.txt
jupyter lab case-01.ipynb
```

В Colab откройте тот же `case-01.ipynb`: первая ячейка скачает ZIP дела с сайта курса, сверит SHA-256 и восстановит ту же структуру каталогов.

## Файлы

- `case-01.ipynb` — рабочая тетрадь с упражнениями;
- `solution/case-01-solution.ipynb` — эталонный разбор;
- `data/SOURCE.md` — происхождение встроенного набора;
- `check_result.md` — ориентиры самопроверки.

Начинайте с рабочей тетради. Решение предназначено для сверки после собственной попытки.
