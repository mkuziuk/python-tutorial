# Ожидаемая форма результата II-06

## Данные
- SHA-256 совпадает с generation_report.json.
- 5 838 строк, 1 797 значений `source_id`, 64 пиксельных столбца.
- Значения `sample_id` уникальны; `instructional_synthetic=True` для каждой строки.
- Присутствуют партии A, B и C, а также значения разбиения `train`, `test`, `holdout`.

## Проверка разбиения поставщика
- Все значения `source_id` из A/B встречаются и в обучающей, и в тестовой выборке поставщика.
- Macro-F1 при построчном разбиении поставщика равна 1,00 для зафиксированной модели.

## Корректная проверка
- В каждой части `StratifiedGroupKFold` пересечение `source_id` между обучением и валидацией равно 0.
- Групповая OOF-оценка macro-F1 — примерно 0,88–0,90.
- Macro-F1 на изолированной до финала партии C — примерно 0,63–0,67.
- Разница между оценкой поставщика и групповой оценкой — не менее 0,05.
- Разница между групповой оценкой и оценкой на партии C — не менее 0,05.
- Показаны показатели по классам, матрица ошибок, галерея ошибок и 95%-й бутстреп-интервал для партии C, рассчитанный пересэмплированием источников.
